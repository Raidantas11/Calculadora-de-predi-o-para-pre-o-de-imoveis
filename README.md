# Calculadora-de-predi-o-para-pre-o-de-imoveis
Uma calculadora que usa modelo preditivo de Machine Learning, capaz de preve o preço de um imovel baseando-se nas variaveis treinadas. No projeto, as variaveis(X) usadas tinham total correlação com o valor do imovel(Y), sendo utilizadas para treinar dois modelos de regressão, um com Random Forest e outro com Regressão Linear.
